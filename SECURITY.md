## Reporting a Vulnerability

If you discover a security vulnerability in Health-Vision, please report it
responsibly by emailing **legal@healthvision.io** with the subject line
`[SECURITY] Vulnerability Report`.

**Do NOT:**
- Create public GitHub issues for security vulnerabilities
- Disclose vulnerabilities publicly before they have been resolved
- Exploit any vulnerability you discover

**Do NOT attempt to:**
- Access systems or data beyond what is necessary to identify the issue
- Perform denial of service attacks
- Conduct social engineering or phishing
- Reverse engineer, decompile, or disassemble any component

We take all security reports seriously and will respond within 72 hours.

---

## Responsible Disclosure Policy

We request that researchers:
1. Report findings privately to legal@healthvision.io
2. Allow a reasonable time (90 days) for remediation before any disclosure
3. Not access, modify, or delete any user data

Researchers acting in good faith and following this policy will not be
subject to legal action by Health-Vision.
